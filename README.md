# Spec-Coding Modulith

## 概要

Spring Modulithを用いて構成されたAPIサーバをSpec駆動開発で実装していく実験用プロジェクト。  
LLMモデルの動作比較や開発用プロンプトの実験を行うためのベースラインとなる資源を提供する。

## 構成

本リポジトリは以下のディレクトリで構成される。

- **backend/**: Spring ModulithベースのJavaバックエンドアプリケーション。
- **doc/**: API仕様書（OpenAPI）、アーキテクチャ設計、ビジネスルール、ユーザーストーリーなどの各種ドキュメントを管理する。
- **portal/**: プロジェクトドキュメントを公開するためのDocusaurusベースのポータルサイト。
